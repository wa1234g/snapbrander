# 📦 SnapBrander - برومبت شامل لبناء نظام SaaS ذكي لتوليد مواقع WordPress باستخدام Laravel + LocalAI + CloudPanel

---

## 🎯 **الهدف العام:**

بناء نظام SaaS متكامل باسم **SnapBrander** يتيح للعملاء إنشاء مواقع WordPress احترافية تلقائياً باستخدام الذكاء الصناعي المحلي (LocalAI)، مع واجهة أمامية مخصصة (ليست WordPress)، وبنية خلفية تعتمد على Laravel، وCloudPanel لإنشاء مواقع مستقلة بملفات وقاعدة بيانات منفصلة.

---

## 🧠 **الواجهة الخلفية (Laravel API)**

### ✅ 1. قاعدة البيانات:

* `users` - معلومات المستخدمين مع دعم الدول واللغات والعملات
* `projects` - مشاريع المواقع مع حالة التجربة والأرشفة
* `plans` - خطط الاشتراك والأسعار
* `templates` - قوالب WordPress مع معاينات وتصنيفات
* `ai_requests` - طلبات الذكاء الاصطناعي وتتبع الاستخدام
* `drafts` - حفظ تقدم النموذج متعدد الخطوات
* `project_logs` - سجل العمليات والأحداث
* `notifications` - إشعارات النظام والتنبيهات
* `template_uploads` - رفع وإدارة القوالب المخصصة
* `session_resume` - استكمال الجلسات المحفوظة
* `team_members` - إدارة الفرق والصلاحيات
* `modules` - الموديولات الإضافية والإضافات
* `project_addons` - ربط المشاريع بالموديولات
* `ai_chat_sessions` - جلسات الدردشة مع المساعد الذكي
* `settings` - إعدادات النظام (العملة، اللغة، المظهر، اللهجة، الموقع الافتراضي، زمن التفعيل)
* `media_gallery` - مكتبة الوسائط والصور
* `marketplace_items` - متجر القوالب والإضافات
* `permissions` - نظام الصلاحيات والأدوار

### ✅ 2. الـ API الأساسية:

#### 🟣 خطوات بناء الموقع:

* `POST /api/projects/init` - بدء مشروع جديد
* `POST /api/ai/generate-description` - توليد وصف المشروع بالذكاء الاصطناعي
* `GET /api/templates` - عرض القوالب المتاحة
* `POST /api/projects/{id}/select-template` - اختيار قالب للمشروع
* `POST /api/ai/generate-colors` - توليد ألوان مناسبة للنشاط
* `POST /api/projects/{id}/set-colors` - حفظ الألوان المختارة
* `POST /api/ai/generate-logo` - توليد شعار بالذكاء الاصطناعي
* `POST /api/projects/{id}/upload-logo` - رفع شعار مخصص
* `POST /api/projects/{id}/generate-site` - إنشاء الموقع النهائي

#### 🟢 الذكاء الصناعي:

* `POST /api/ai/generate-content` - توليد محتوى الصفحات
* `POST /api/ai/translate` - ترجمة المحتوى
* `POST /api/ai/chat` - مساعد ذكي خطوة بخطوة

#### 🟠 إضافات مستقبلية:

* `POST /api/projects/{id}/install-addon` - تثبيت إضافة
* `GET /api/modules` - قائمة الموديولات الاختيارية

## 🧑‍💻 **الواجهة الأمامية (Frontend React)**

### ✅ 1. نموذج متعدد الخطوات:

* **معلومات النشاط** - اسم الشركة، نوع النشاط، الصناعة
* **وصف المشروع** - وصف تفصيلي أو توليد ذكي
* **اختيار نوع الموقع** - شركة / متجر / صفحة هبوط
* **اختيار القالب** - معرض القوالب مع معاينات
* **اختيار الألوان** - لوحات ألوان مولدة بالذكاء الاصطناعي
* **رفع/توليد الشعار** - رفع شعار أو توليد بالذكاء الاصطناعي
* **اختيار الموديولات الإضافية** - نموذج تواصل، دردشة، حجز، سوشيال ميديا
* **زر إنشاء الموقع الآن** - المراجعة النهائية والإنشاء

### ✅ 2. المعاينة والتفاعل:

* معاينة القالب داخل iframe أو صورة متحركة
* مساعد ذكي (chat widget) داخل كل خطوة
* تصميم مبهر باستخدام خط Cairo
* دعم RTL للغة العربية
* اختيار الدولة في نموذج التسجيل

### ✅ 3. استكمال النموذج:

* حفظ تلقائي لكل خطوة في `draft`
* عند تسجيل دخول جديد → الرجوع للخطوة السابقة من `session_resume`
* إدارة حالة الـ wizard بكفاءة

## 🔄 **دورة حياة المشروع (Lifecycle):**

1. **البداية**: يبدأ العميل تعبئة النموذج
2. **الحفظ التلقائي**: يتم حفظ كل خطوة في `draft`
3. **اختيار القالب**: يتم حفظه في `projects`
4. **إنشاء الموقع**: عند الضغط على "إنشاء موقعي":
   * توليد دومين فرعي تلقائياً (project-xyz.fureraa.com)
   * يتم إنشاء موقع جديد على CloudPanel (مستقل وليس multisite)
   * تثبيت WordPress + Hello Elementor
   * إذا كان متجر: تثبيت WooCommerce
   * استيراد قالب TemplateKit
   * تخصيص الألوان والشعار والمحتوى تلقائياً باستخدام LocalAI
5. **التسليم**: يعرض النظام بيانات الدخول + رابط الموقع
6. **التجربة**: يبدأ العداد التنازلي لمدة 72 ساعة
7. **الأرشفة**: إذا لم يتم الاشتراك → يتم نقل الموقع للأرشيف ثم الحذف بعد 3 أيام
